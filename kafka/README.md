## Projects

- [kafka_demo](https://github.com/Rigbir/backend_tasks/tree/main/kafka/kafka_demo) - Kafka Producer and Consumer implementation with KRaft mode
- [kafka_async](https://github.com/Rigbir/backend_tasks/tree/main/kafka/kafka_async) - Asynchronous Kafka Producer and Consumer with worker threads
